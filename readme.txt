Student Management Readme

作者：沈洪斌
博客：http://www.cnblogs.com/vincenshen/

程序介绍：
	后端基于Django web框架
	前端基于bootstrap框架
	数据库为sqlite3
	
实现功能：
	简单用户管理：
		用户注册
		用户登录
	Student信息的增删改查
	
注意事项：
	Django版本为1.11.x
	
使用方法：
	启动服务端：pyhton manage.py  runserver
	首页地址： http://127.0.0.1/students
